## 🏛️ Corporate Identity
**Gulf Business Solutions (GBS)** is the KSA-based entity of CNS — an ISO 9001:2015, ISO/IEC 20000-1:2018 & ISO/IEC 27001:2022 certified BIM consulting firm. As a strategic **Autodesk Silver Partner**, GBS delivers digital transformation for the Middle East's most ambitious Giga-projects (Diriyah Gate, NEOM, etc.).

## ⚛️ Technical Stack
The GBS website is a high-performance **Next.js 15** application designed with a **"Future-Built"** aesthetic:

- **Frontend**: Next.js 15 (App Router) + React 19
- **Styling**: Tailwind CSS 4 with custom **Cyan/Blue Glassmorphism**
- **Animations**: Framer Motion for premium micro-interactions
- **3D Engine**: React Three Fiber (R3F) + Three.js for interactive logos and shaders
- **Deployment**: Automated CI/CD to GitHub Pages via Static Export

## ✨ Visual Modernization Features
- **3D Hero Intro**: Dynamic interactive disk with high-fidelity GBS logo integration.
- **Glassmorphic UI**: Premium `backdrop-blur-2xl` components throughout for a sleek corporate feel.
- **Cyan/Blue Palette**: Standardized `#00ADEF` / `#1282C4` branding.
- **BIM ROI Calculator**: Interactive financial planning tool for AEC stakeholders.
- **Landmark Showcases**: High-contrast, high-fidelity project cards for flagship projects.
